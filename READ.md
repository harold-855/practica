# Rock Band Landing Page

## Description

This project is a responsive landing page for a rock band.
The purpose of the website is to promote the band, present upcoming concerts, highlight music and experiences, and create a visually attractive interface for fans.

The page focuses on semantic HTML structure and modern CSS layout techniques, ensuring readability, responsiveness, and a clean user experience.

---

## Technologies Used

* HTML5 (Semantic Structure)
* CSS3 (Flexbox, Grid Layout, Media Queries)
* JavaScript (Basic interactivity)

---

## Project Structure

```bash
project-folder/
│
├── index.html          # Main HTML file
├── script.js           # JavaScript for interactivity
├── styles.css          # Styles for the page
├── assets/
│   └── img/            # Images used in the project
├── pages/
│   └── section.html    # Additional page sections
└── README.md           # Project documentation
```

---

## Features

* Semantic HTML structure (header, main, section, footer)
* Hero section with strong visual impact and CTA
* Events section with structured HTML table (thead, tbody)
* About section with descriptive content
* Unordered list (ul) for band experiences
* Multimedia gallery using CSS Grid
* Responsive layout using media queries
* Navigation adapted for mobile view
* Clean and organized UI

---

## Responsive Design

The page adapts to different screen sizes:

* Mobile devices
* Desktop screens (>1024px)

Media queries are used to adjust:

* Layout (Grid/Flex)
* Typography
* Spacing
* Navigation

2. Open the project folder.
