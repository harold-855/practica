# Festival Gastronómico

## Descripción

Este proyecto es una página web para un festival gastronómico nocturno. El objetivo es mostrar el evento, la zona de comidas y la programación de forma clara y adaptable a móviles.

## Estructura del proyecto

- `index.html`: página principal con la estructura del contenido.
- `css/style.css`: archivo principal que importa todos los estilos.
- `css/header.css`: estilos del encabezado y navegación.
- `css/hero.css`: estilos de la sección principal de bienvenida.
- `css/table.css`: estilos para la tabla de eventos.
- `css/card.css`: estilos para las tarjetas de zonas del festival.
- `css/responsive.css`: estilos responsivos para móviles y tablets.
- `js/welcome.js`: comportamiento interactivo del menú y mensaje.

## Comentarios clave

- Usa `meta viewport` para que la página sea responsive en móviles.
- La tabla se envuelve en un contenedor con `overflow-x: auto` para que no se rompa en pantallas pequeñas.
- Las tarjetas usan `grid` con `auto-fit` para adaptarse al ancho disponible.

## Cómo abrir la página

1. Abre `index.html` en tu navegador.
2. Si usas Visual Studio Code, instala Live Server y abre el archivo con Live Server.
